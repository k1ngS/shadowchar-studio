# Logo — Shadowchar Studio

## Arquivos
- shadowchar-logo-v4.jpg   → logo principal (símbolo + wordmark)
- shadowchar-favicon.png   → ícone isolado para favicon

## Uso
- Fundo sempre escuro (#050507 ou transparente)
- Nunca inverter cores
- Espaçamento mínimo ao redor: 16px
- Não redimensionar abaixo de 120px de largura para o logo completo
- Favicon: usar apenas o símbolo (shard) sem o wordmark

## Tipografia do wordmark
Cinzel — Google Fonts (https://fonts.google.com/specimen/Cinzel)
